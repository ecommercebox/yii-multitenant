Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist solutosoft/yii-multitenant
```

or add

```json
"solutosoft/yii-multitenant": "*"
```
